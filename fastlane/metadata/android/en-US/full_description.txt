<p>LibreCrate is a secure document vault that stores, views, organizes, and searches PDFs, EPUBs, Apple Wallet passes (PKPass), comic archives (CBZ/CBR), images, and personal Markdown notes — all encrypted at rest.</p>

<p>All documents are encrypted with AES-256-GCM. An optional password wraps the master key via Argon2id + AES-256-KW, so content stays protected even when the phone is unlocked. There is no internet permission — documents never leave the device.</p>

<p>Features:</p>
<ul>
<li>Six supported types: PDF, EPUB, PKPass, CBZ/CBR, images, and notes</li>
<li>Per-file AES-256-GCM encryption</li>
<li>Optional password protection (Argon2id key derivation)</li>
<li>PDF viewer with pinch-to-zoom, smooth vertical scrolling, and pixel-perfect page rendering</li>
<li>Search, sort, and filter library with reading progress indicators</li>
<li>Share intent and SAF file picker import</li>
<li>Encrypted backup export/import via SAF</li>
<li>Zero network access — no Google Play Services, Firebase, or ads</li>
</ul>
